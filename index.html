Programação de sistemas distribuídos - JAVA

1. Conceitos Básicos e Plataforma Java

Java é uma linguagem de programação de alto nível, orientada a objetos e multiplataforma. Foi desenvolvida para ser portátil e segura, permitindo que aplicações sejam executadas em qualquer sistema que possua a Java Virtual Machine (JVM). Java é amplamente utilizado para aplicações web, móveis e empresariais.

O código abaixo mostra um simples programa Java que imprime "Olá, Mundo!":

public class Main {
    public static void main(String[] args) {
        System.out.println("Olá, Mundo!");
    }
}

2. Fundamentos da Programação Orientada a Objetos

A Programação Orientada a Objetos (POO) é um paradigma de programação que se baseia no conceito de "objetos", que representam entidades do mundo real. Esses objetos possuem atributos (dados) e métodos (comportamentos). Java adota esse modelo para tornar o código mais modular e reutilizável.

O código abaixo mostra a estrutura básica de uma classe e um objeto em Java:

class Pessoa {
    String nome;
    int idade;
    
    void apresentar() {
        System.out.println("Olá, meu nome é " + nome + " e tenho " + idade + " anos.");
    }
}

3. Tipos, Literais, Operadores e Controlo de Fluxo

Java possui vários tipos de dados, operadores e estruturas de controle de fluxo para gerenciar a lógica do programa. Alguns exemplos:

  • Tipos: int, double, boolean, char, String.
  • Operadores: +, -, *, /, %, ==, !=, &&, ||.
  • Estruturas de controle: if-else, switch, loops (for, while, do-while).
int idade = 20;
if (idade >= 18) {
    System.out.println("Maior de idade");
} else {
    System.out.println("Menor de idade");
}

4. Criação de Classes, Objetos, Métodos e Variáveis

Em Java, as classes são usadas para definir o comportamento dos objetos. Cada classe pode ter variáveis de instância e métodos para manipular esses dados.

class Carro {
    String modelo;
    int ano;
    
    void exibirDetalhes() {
        System.out.println("Modelo: " + modelo + ", Ano: " + ano);
    }
}

5. Reutilização com Herança e Composição

A herança permite que uma classe filha herde atributos e métodos de uma classe pai. Já a composição reutiliza funcionalidades incluindo objetos dentro de outras classes.

class Animal {
    void emitirSom() {
        System.out.println("Som genérico de animal");
    }
}

class Cachorro extends Animal {
    void emitirSom() {
        System.out.println("Au au!");
    }
}

6. Interfaces e Polimorfismo

Interfaces definem um conjunto de métodos que uma classe deve implementar. O polimorfismo permite que métodos tenham comportamentos diferentes dependendo da classe.

interface Animal {
    void fazerSom();
}

class Gato implements Animal {
    public void fazerSom() {
        System.out.println("Miau!");
    }
}

7. Classes Internas

Classes internas são utilizadas para melhorar a organização do código e encapsular funcionalidades específicas dentro de uma classe maior.

class Externa {
    int x = 10;

    class Interna {
        void mostrar() {
            System.out.println("Valor de x: " + x);
        }
    }
}

8. Pacotes, Encapsulamento e JavaBeans

Os pacotes organizam classes em módulos lógicos. O encapsulamento protege os dados, permitindo acesso apenas por métodos controlados. JavaBeans são classes que seguem um padrão específico para reutilização eficiente.

9. Coleções e Strings

O Java oferece coleções como ArrayList e HashMap para manipulação eficiente de dados, além de classes para manipulação de strings.

import java.util.ArrayList;
ArrayList lista = new ArrayList<>();
lista.add("Maçã");
lista.add("Banana");
System.out.println(lista.get(0));

10. Aplicações Gráficas em Java

Com a biblioteca Swing, é possível criar interfaces gráficas para aplicações Java.

import javax.swing.*;
JFrame frame = new JFrame("Minha Janela");
frame.setSize(400, 300);
frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
frame.setVisible(true);

11. Entrada e Saída (java.io)

A API java.io permite ler e escrever arquivos.

import java.io.*;
BufferedReader leitor = new BufferedReader(new FileReader("texto.txt"));
String linha;
while ((linha = leitor.readLine()) != null) {
    System.out.println(linha);
}
leitor.close();

12. Controlo de Erros e Exceções

Erros podem ser tratados usando blocos try-catch para evitar falhas no programa.

try {
    int resultado = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Erro: divisão por zero!");
} finally {
    System.out.println("Execução finalizada.");
}
